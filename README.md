# Notelang
## simple note taking language that compiles to html

## Usage
---
`notelang myfile.ntlang` <br>
(generates `out.html`)

## Basic tags
---

### normal text
`.p This is a normal line of text.`

### headlines
`.h Some headline`

### bold text
`some * bold * text`

### italic text
`some # italic # text`

### marked text
`some _ marked _ text`

### images
`.img Image.png`
